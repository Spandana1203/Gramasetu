Diksha Platform (Digital Learning)

Description: DIKSHA, launched in 2017 by NCERT, is India's national digital platform for school education. It provides educational resources for teachers, students, and parents in 36+ Indian languages via web, Android, and iOS.

Benefits

  • Teachers: Professional development courses, lesson planning tools, personalized dashboards, participate in VidyaDaan initiative
  • Students: Interactive lessons, videos, worksheets, NCERT textbooks, self-assessment tools
  • Parents: Tools to monitor children's learning progress, access resources for home learning support

Eligibility

  • Teachers: NCERT, CBSE, SCERT or recognized board school educators
  • Students: Enrolled in schools across India
  • Parents: Guardians supporting children‚Äôs education

How to Apply / Access

  • Visit the DIKSHA Portal: https://diksha.gov.in/
  • Select your user role ‚Äì Teacher / Student / Parent
  • Register or Login
  • Set preferences: board, language, class/grade
  • Explore resources: lessons, courses, tools

Documents Required

  • Teachers: School ID / Teacher ID, Aadhaar, mobile linked to Aadhaar
  • Students: Student ID or enrollment number, Aadhaar (optional), parent's mobile
  • Parents: Aadhaar (optional), mobile linked to Aadhaar
